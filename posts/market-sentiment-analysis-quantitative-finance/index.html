引言

传统量化因子（动量、价值、波动率）主要基于价量数据。然而，市场参与者的情绪和预期往往先于价格变动。将非结构化文本转化为结构化情绪信号，是 Alternative Data 在量化投资中最成熟的应用之一。

4. 回测框架

import pandas as pd
import numpy as np

class SentimentBacktest:
    def __init__(self, universe: pd.Index, lookback: int = 5):
        self.universe = universe
        self.lookback = lookback

    def compute_factor(self, sentiment_scores: pd.DataFrame) -> pd.DataFrame:
        """Convert raw sentiment to tradeable factor."""
        # EWMA smoothing
        smoothed = sentiment_scores.ewm(span=self.lookback).mean()
        # Cross-sectional z-score
        z = smoothed.sub(smoothed.mean(axis=1), axis=0).div(
            smoothed.std(axis=1), axis=0
        )
        # Winsorize at ±3σ
        return z.clip(-3, 3)

    def evaluate_ic(
        self, factor: pd.DataFrame, forward_returns: pd.DataFrame
    ) -> pd.Series:
        """Compute daily rank IC."""
        ic = factor.corrwith(forward_returns, axis=1, method="spearman")
        return ic

    def summary(self, ic_series: pd.Series) -> dict:
        return {
            "IC_mean": ic_series.mean(),
            "IC_std": ic_series.std(),
            "ICIR": ic_series.mean() / ic_series.std(),
            "IC_positive_ratio": (ic_series > 0).mean(),
        }

5. 实践中的陷阱

  1. 前视偏差 (Look-Ahead Bias): 新闻发布时间 ≠ 数据可用时间。必须使用 point-in-time 数据库。
  2. 幸存者偏差: 退市股票的情绪数据缺失会高估因子表现。
  3. 过拟合: 情绪因子的 IC 通常较低（0.02-0.05），容易在样本内过拟合。使用 purged k-fold CV：
Traink={t:t∉[Testkstart−embargo,Testkend]}\text{Train}_k = \{t : t \notin [\text{Test}_k^{\text{start}} - \text{embargo}, \text{Test}_k^{\text{end}}]\}Traink​={t:t∈/[Testkstart​−embargo,Testkend​]}
  1. 容量约束: 高频情绪信号在大资金规模下 Alpha 衰减显著。

总结

市场情绪分析的核心挑战不在 NLP 模型本身，而在于信号到因子的工程化——包括数据时效性、因子正交化、以及与现有因子体系的融合。一个 IC=0.03 但稳定且与传统因子低相关的情绪因子，在组合层面的贡献可能远超其单因子表现。
