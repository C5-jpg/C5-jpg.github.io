图神经网络 (GNN) 核心机制：从消息传递到注意力聚合

深入解析 GNN 的消息传递范式、邻域聚合策略以及 Graph Attention Network 的注意力机制，附关键数学推导。

引言

图神经网络 (Graph Neural Networks, GNN) 是处理非欧几里得结构数据的核心工具。与传统 CNN 在规则网格上卷积不同，GNN 需要在任意拓扑结构上定义”卷积”操作。本文从第一性原理出发，拆解 GNN 的核心计算范式。

1. 消息传递神经网络 (MPNN) 框架

Gilmer et al. (2017) 提出的 MPNN 框架将几乎所有 GNN 变体统一为两步操作：

消息聚合 (Message Aggregation):

mv(l+1)=⨁u∈N(v)M(l)(hv(l),hu(l),evu)m_v^{(l+1)} = \bigoplus_{u \in \mathcal{N}(v)} M^{(l)}\left(h_v^{(l)}, h_u^{(l)}, e_{vu}\right)mv(l+1)​=u∈N(v)⨁​M(l)(hv(l)​,hu(l)​,evu​)

节点更新 (Node Update):

hv(l+1)=U(l)(hv(l),mv(l+1))h_v^{(l+1)} = U^{(l)}\left(h_v^{(l)}, m_v^{(l+1)}\right)hv(l+1)​=U(l)(hv(l)​,mv(l+1)​)

其中 ⨁\bigoplus⨁ 是置换不变的聚合函数（如 sum、mean、max），M(l)M^{(l)}M(l) 是消息函数，U(l)U^{(l)}U(l) 是更新函数。

2. GCN: 谱域到空间域的桥梁

Kipf & Welling (2017) 的 GCN 通过一阶切比雪夫近似简化了谱图卷积：

H(l+1)=σ(D~−12A~D~−12H(l)W(l))H^{(l+1)} = \sigma\left(\tilde{D}^{-\frac{1}{2}} \tilde{A} \tilde{D}^{-\frac{1}{2}} H^{(l)} W^{(l)}\right)H(l+1)=σ(D~−21​A~D~−21​H(l)W(l))

其中 A~=A+IN\tilde{A} = A + I_NA~=A+IN​ 是加入自环的邻接矩阵，D~ii=∑jA~ij\tilde{D}_{ii} = \sum_j \tilde{A}_{ij}D~ii​=∑j​A~ij​ 是度矩阵。

关键洞察： 对称归一化 D~−1/2A~D~−1/2\tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}D~−1/2A~D~−1/2 等价于对邻居特征做加权平均，权重与节点度的几何平均成反比。

3. GAT: 注意力驱动的自适应聚合

Graph Attention Network (Veličković et al., 2018) 引入可学习的注意力系数：

αvu=exp⁡(LeakyReLU(aT[Whv∥Whu]))∑k∈N(v)exp⁡(LeakyReLU(aT[Whv∥Whk]))\alpha_{vu} = \frac{\exp\left(\text{LeakyReLU}\left(\mathbf{a}^T [W h_v \| W h_u]\right)\right)}{\sum_{k \in \mathcal{N}(v)} \exp\left(\text{LeakyReLU}\left(\mathbf{a}^T [W h_v \| W h_k]\right)\right)}αvu​=∑k∈N(v)​exp(LeakyReLU(aT[Whv​∥Whk​]))exp(LeakyReLU(aT[Whv​∥Whu​]))​

节点更新变为：

hv(l+1)=σ(∑u∈N(v)αvuWhu(l))h_v^{(l+1)} = \sigma\left(\sum_{u \in \mathcal{N}(v)} \alpha_{vu} W h_u^{(l)}\right)hv(l+1)​=σ​u∈N(v)∑​αvu​Whu(l)​​

多头注意力进一步增强表达能力：

hv(l+1)=∥k=1Kσ(∑u∈N(v)αvukWkhu(l))h_v^{(l+1)} = \Big\|_{k=1}^{K} \sigma\left(\sum_{u \in \mathcal{N}(v)} \alpha_{vu}^k W^k h_u^{(l)}\right)hv(l+1)​=​k=1K​σ​u∈N(v)∑​αvuk​Wkhu(l)​​

4. 过平滑问题与深层 GNN

随着层数增加，所有节点表示趋于收敛（over-smoothing）。定量地，lll 层 GCN 的感受野为 lll-hop 邻域，当 l→∞l \to \inftyl→∞ 时：

lim⁡l→∞H(l)∝1⋅πT\lim_{l \to \infty} H^{(l)} \propto \mathbf{1} \cdot \pi^Tl→∞lim​H(l)∝1⋅πT

其中 π\piπ 是图上随机游走的平稳分布。

缓解策略：

  • 残差连接： hv(l+1)=hv(l+1)+hv(l)h_v^{(l+1)} = h_v^{(l+1)} + h_v^{(l)}hv(l+1)​=hv(l+1)​+hv(l)​
  • DropEdge： 训练时随机移除边
  • PairNorm / NodeNorm： 层间归一化

5. 代码示例：PyG 中的 GAT

import torch
from torch_geometric.nn import GATConv

class GATModel(torch.nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim, heads=8):
        super().__init__()
        self.conv1 = GATConv(in_dim, hidden_dim, heads=heads, dropout=0.6)
        self.conv2 = GATConv(hidden_dim * heads, out_dim, heads=1, concat=False, dropout=0.6)

    def forward(self, x, edge_index):
        x = torch.nn.functional.elu(self.conv1(x, edge_index))
        x = self.conv2(x, edge_index)
        return x

总结

GNN 的核心在于局部聚合 + 全局传播。理解 MPNN 框架后，GCN、GAT、GraphSAGE 等变体只是在消息函数 MMM 和聚合算子 ⨁\bigoplus⨁ 上的不同设计选择。下一步值得关注的方向包括异构图 Transformer 和 GNN 的表达力理论边界（Weisfeiler-Leman 层次）。
